# KET单词闪卡网站 需求规格说明

## 项目背景
基于剑桥KET核心词汇文档（黑1/蓝2/红3三个级别），制作一个卡通风格的单词闪卡学习网站。

## 核心功能需求
### 1. 单词闪卡功能
- 每次展示一个单词闪卡，包含：
  - 单词上方显示音标，音节标记清晰
  - 单词本身尽可能大字体显示
  - 中文释义
  - 级别标签（黑1/蓝2/红3），用对应颜色显示（黑色/蓝色/红色）
  - 主题标签（如动物、人物、衣服等，从Word文档中提取）
  - 单词旁边有播放按钮▶️，点击可发音，兼容Chrome、360等主流浏览器内核
- 支持两种学习模式切换：
  - 按级别记忆：黑1 → 蓝2 → 红3，阶梯升级难度
  - 按主题记忆：按单词分类（动物/人物/衣服等）分类学习
- 支持手动点击进入下一个单词
- 用户点击「会」或「不会」后自动跳转到下一个单词
- 单词切换有自然流畅的过渡动画，避免生硬跳转
- 整体UI为卡通风格
- 首屏优先加载，保证加载速度

### 2. 单词本功能
- 支持标记单词为「已会」或「不会」
- 「已会」单词计入已会单词本
- 「不会」单词计入不会单词本
- 可查看不同分类的单词本

### 3. 统计功能
- 显示单词总数量
- 显示当前学习进度
- 显示已会单词数量
- 显示不会单词数量
- 学习进度和单词本数据支持本地持久化存储，刷新页面或下次打开后数据不丢失，可继续上次进度学习

### 4. 导出功能
- 支持将单词本导出为PDF文档

### 5. 背景切换功能
- 提供多种背景主题可选：森林、海洋、沙漠等
- 用户可自由切换背景

## 数据源
- 词汇来源：`/home/admin/.openclaw/workspace-codingagent/projects/剑桥1-3单词.docx`
- 词汇分级：黑1（一级，黑色标签）、蓝2（二级，蓝色标签）、红3（三级，红色标签）
- 词汇分类：按主题分类（动物、人物、衣服等，从Word文档中提取）

## 非功能需求
- 响应式设计，支持PC和移动端访问
- 界面美观，卡通风格，适合学生使用
- 加载速度快，操作流畅
